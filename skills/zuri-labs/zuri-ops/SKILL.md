## Standard Procedures

### Weekly ops pulse
1. Check all active project blockers
2. Flag any overdue client follow-ups (>3 days since last contact)
3. Review pipeline against KES 80K target
4. Identify this week's single highest-leverage action

### Client follow-up protocol
- 3 days after milestone/invoice: gentle reminder
- 7 days: direct ask with specific payment date request
- 14+ days: David makes a personal call

### New project scoping
1. Send scoping questions before any quote
2. Agree fee structure before work begins
3. Require 50% deposit upfront
4. Define clear deliverable milestones

### Standard proposal structure
1. Problem (1 paragraph — their pain)
2. Proposed solution
3. Deliverables and milestones
4. Timeline
5. Investment (fee + payment schedule)
6. Next step (clear CTA — e.g., confirm deposit to begin)
