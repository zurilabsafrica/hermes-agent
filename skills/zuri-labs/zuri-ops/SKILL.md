## Standard Procedures

### Weekly ops pulse format
```
ACTIVE PROJECTS
- [project]: [status] — [next action]

CLIENT HEALTH
- [name]: [health] — [action needed]

PIPELINE
- [prospect]: [stage] — [next step]

URGENT THIS WEEK
1. [single highest-leverage action]
```

### New project intake
1. Send scoping questions before any quote
2. Agree fee structure before work begins
3. Require 50% deposit upfront
4. Define clear deliverable milestones

### Client follow-up protocol
- Day 3 after invoice or milestone: gentle reminder
- Day 7: direct ask — request specific payment date
- Day 14+: David makes personal call or message

### Standard proposal structure
1. Problem — their pain (1 paragraph)
2. Proposed solution
3. Deliverables and milestones
4. Timeline
5. Investment (fee + payment schedule)
6. Next step — clear CTA (e.g., confirm deposit to begin)
